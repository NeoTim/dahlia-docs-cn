### 初始化

如果你没有安装 Dahlia CLI，请看 [安装 Dahlia CLI](https://github.com/forsigner/dahlia-docs/tree/a642eb4ce505b4ada4123290ab6f37ef3da03bad/docs-getting-started/README.md#-第一步-安装-dahlia-cli)。

使用 Dahlia CLI 初始化应用:

```bash
dh new friends
```

初始化成功后，启动本地开发服务器：

```bash
cd friends
dh dev
```
